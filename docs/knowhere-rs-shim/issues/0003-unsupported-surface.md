# 0003 Unsupported Surface

## Explicitly Out of Stage 1

- `RangeSearch`
- `GetVectorByIds`
- iterators
- IVF family
- DiskANN
- sparse indexes
- GPU paths
- `mmap` / disk-load semantics

## Handling Rule

- If Milvus requires one of these surfaces to compile or run the smoke lane, stop and add a focused follow-up issue instead of silently expanding scope
